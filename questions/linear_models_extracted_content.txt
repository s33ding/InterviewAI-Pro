## 2. Matriz Determinante (Matrix Determinant)
Key Concepts:
• **Definition:** A scalar value associated with square matrices that 
summarizes essential properties
• **Invertibility Test:** Matrix is invertible if and only if 
determinant ≠ 0
• **Geometric Interpretation:** Measures scaling factor of 
transformations (area/volume changes)
• **Conceptual Meaning:** Indicates "non-degeneracy" - zero determinant 
means matrix "flattens" space

Educational Applications:
• Testing matrix invertibility
• Understanding linear transformations
• Geometric interpretations of matrix operations

## 3. Matriz Identidade (Identity Matrix)
Key Concepts:
• **Structure:** Square matrix with 1s on main diagonal, 0s elsewhere
• **Property:** Neutral element for matrix multiplication (A × I = A)
• **Notation:** Typically denoted as I_n for n×n identity matrix

Educational Applications:
• Foundation for understanding matrix operations
• Basis for matrix inversion concepts
• Essential for linear algebra proofs

## 4. Covariance
Key Concepts:
• **Purpose:** Measures how two variables change together
• **Interpretation:** 
  • Positive: variables increase/decrease together
  • Negative: one increases while other decreases
• **Formulas:**
  • Population: Uses μ (population means)
  • Sample: Uses sample means with N observations
• **Applications:** Used in PCA and other statistical techniques

Educational Applications:
• Understanding variable relationships
• Foundation for correlation analysis
• Prerequisite for dimensionality reduction techniques

## 5. PCA (Principal Component Analysis)
Key Concepts:
• **Purpose:** Linear dimensionality reduction technique
• **Method:** Projects high-dimensional data into lower-dimensional 
subspace
• **Mathematical Foundation:** Finding eigenvectors of covariance matrix
• **Goal:** Identify directions of maximum variance in data
• **Type:** Unsupervised learning technique
• **Applications:** Data clustering without labels

Educational Applications:
• Dimensionality reduction in machine learning
• Data visualization and compression
• Feature extraction and noise reduction

## 6. Pearson Correlation
Key Concepts:
• **Purpose:** Measures strength and direction of linear relationships
• **Range:** -1 to 1
• **Interpretation:**
  • +1: Perfect positive linear relationship
  • -1: Perfect negative linear relationship
  • 0: No linear relationship
• **Difference from Covariance:** Standardized (unitless) measure
• **Formula:** Standardizes covariance by dividing by standard 
deviations

Educational Applications:
• Statistical analysis of variable relationships
• Data exploration and hypothesis testing
• Quality assessment in predictive modeling

## 7. Spearman Correlation
Key Concepts:
• **Purpose:** Assesses monotonic relationships using ranked values
• **Type:** Non-parametric correlation measure
• **Applications:** Ordinal data and non-linear but consistent trends
• **Formula:** Based on rank differences (d_i) between observations
• **Advantage:** Works with non-linear monotonic relationships

Educational Applications:
• Analysis of ordinal data
• Robust correlation analysis
• Alternative to Pearson when assumptions are violated
