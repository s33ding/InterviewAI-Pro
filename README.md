InterviewAI-Pro

InterviewAI-Pro is an AI-powered technical interview prep platform, built using serverless architecture on AWS. It leverages Lambda, S3, Cognito, Lex, and Bedrock to deliver a scalable, efficient, and secure interview preparation experience. Currently supporting Software Engineering, Data Science, Machine Learning, and System Design, users receive AI-driven feedback, track progress, and refine their skills with real-time insights.
